# 就活メモ一括管理ツール【完全版】

自身の就職活動を効率化し、自己分析や企業研究のデータを一元管理するために開発した、ブラウザ上で動作するWebアプリケーションです。
本プロジェクトは、AIアシスタント **Google Gemini** を利用してバイブコーディングにより作成しました。
今後もUXの改善やスケジュール管理機能など新機能の実装などを行い、より就職活動を効率化できるWebアプリケーションにしていく予定です。

## 開発の背景と目的
就職活動を進める中で自己分析の内容や企業分析、提出したESで書いた内容をメモしても、それぞれ媒体が異なっていたり、メモアプリでは他のメモと混在していたりして、どこにメモをしたかがすぐに分かりませんでした。
そのため、就職活動を効率化するために、どこでも手軽に就活に関することのみを一元的に管理できる機能を持つWebアプリケーションを作りたいと考えました。

## 主な機能
- **カテゴリでのメモの分類**: 「大分類（自己分析/業界/企業）」＞「中分類（企業名など）」＞「小分類（ES/面接など）」で整理。
- **直感的なドラッグ＆ドロップでの並び替え機能**: メモのタブを長押しして、優先度順に直感的に並び替えができる。
- **削除＆復元機能（ゴミ箱機能）**: 誤消去を防ぐアラート機能と、サイドバーからのワンクリック復元機能を搭載。
- **データバックアップ＆復元機能（JSONエクスポート/インポート）**: 
  ブラウザのローカルストレージ（Local Storage）を活用し、データを強固に保持。さらに、週次でのバックアップやPC移行に対応するため、データを1つのJSONファイルとしてローカルに保存・復元できる機能を実装。
- **画像添付＆サイズ調整機能**: 企業ロゴや参考資料のスクリーンショットを貼り付け、スライダーで好みの幅に調整可能。

## 技術スタック / 開発体制
- **フロントエンド**: HTML5 / CSS3 (Modern Flexbox, CSS Variables) / JavaScript (Vanilla JS, HTML5 Drag and Drop API)
- **データ管理**: Web Storage API (Local Storage)
- **開発体制**: **Google Gemini (AI)**
  - Geminiに実装したい機能の要件を提示し、ベースとなるコードを出力。
  - 実際の動作テストを繰り返し、エラー発生時の原因究備（デバッグ）や、追加機能の提案をGeminiから受けながら、ツールのブラッシュアップを重ねて完成させました。

## 導入・使用方法
1. 本リポジトリから `portfolio-gemini-code-note.html` をダウンロードします。
2. 任意のフォルダに配置し、ファイルをGoogle Chrome等のブラウザにドラッグ＆ドロップするだけで、ログイン不要で即座に使用可能です。
